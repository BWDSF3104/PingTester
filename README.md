# PingTester

オンラインゲームのプレイヤー間における**インターネット経由**の遅延（RTT）を確認するツールです。

## 注意事項

- **同一ネットワーク（LAN）内での使用は非対応です。**  
  本ツールの用途はオンラインゲームにおける回線遅延の確認であり、同一ルータ配下の端末間計測は対象外としています。  
  同一 NAT 環境では外部 IP アドレスが一致するため、相手エントリが自動登録されません。

## 使い方

1. 各自が `Settings.xml` の `RoomId` に共通の文字列を設定して起動する
2. MQTT 経由で互いの外部 IP:Port が自動交換される
3. 「開始」ボタンで UDP Ping 計測を実行する
